### Changed
- Improved pre-configured keybindings
- BetterF3 configuration
- CommandKeys configuration
- Extreme sound muffler configuration to include the sound of explosive arrows emitted during mob waves in the factory
- Sodium Extra configuration
- ModMenu is now accessible by pressing RSHIFT
- "/cc coins" can't be accessed by pressing SHIFT + F anymore. This fixes a keybind conflict. The command has not been reassigned to another keybind as the amount of coins is always rendered on the screen anyways.
- Removed coordinate display in the top left
- Added coordinate display next to Xaero's Minimap
- Rearranged CCLive-Utilities GUI elements

### Fixed
- Fixed keybind conflicts
- Fixed an issue with title screen rendering due to server resource pack caching
