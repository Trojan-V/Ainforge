# Ainforge

Ainforge is a Minecraft **Fabric** modpack for Minecraft **1.21.8** (Fabric Loader **0.18.2**) tailored for playing *Cactus Clicker* on the PlayLegend server network (**playlegend.net**).   
It bundles pre-configured client-side mods to improve performance and provide a more seamless, PlayLegend-themed experience. 

## Download

- Modrinth: https://modrinth.com/modpack/ainforge 
- Source / pack maintenance repo: https://github.com/Trojan-V/Ainforge 

## Goals

- Smooth, stable gameplay on PlayLegend (playlegend.net), especially for the Cactus Clicker mode. 
- A seamless look & feel (including a custom title screen). 
- Convenience features and client-side improvements that make the Cactus Clicker experience better. 

## What’s inside

This modpack focuses on:
- Performance/optimization mods (FPS improvements, reduced stutter, faster loading). 
- A custom title screen themed after PlayLegend. 
- Cactus-Clicker-specific helper mods that add/enable features relevant to that mode. 

## Installation

1. Install the Prism Launcher (recommended) or a launcher that supports Modrinth modpacks.
2. Create a new instance, select Modrinth and install the pack by searching for "Ainforge".
3. Start Minecraft and join `playlegend.net`. 

## Compatibility

- Minecraft: 1.21.8 
- Mod loader: Fabric (Loader 0.18.2) 
- Intended usage: Client-side on the PlayLegend network (playlegend.net). 

## Keybindings

Below are all keybindings that are **pre-configured** in Ainforge and differ from Minecraft defaults, grouped by mod. 

### CCLive-Utilities
- **F6**: Open Overlay Editor 
- **F7**: Toggle Material Tracker 
- **F8**: Toggle Cards & Statues 
- **F9**: Toggle Kill Tracker 
- **F10**: Toggle Blueprint Tracker 
- **F12**: Toggle Smithing State Frames 

### Fabrishot
- **CTRL + SHIFT + F2**: Take large screenshot 

### GammaUtils
- **Down Arrow**: Decrease Gamma 
- **Up Arrow**: Increase Gamma 
- **G**: Toggle Gamma 

### Iris
- **K**: Toggle Shaders 
- **O**: Shaderpack Selection Screen 

### ModMenu
- Right Shift (**RSHIFT**): Open Mod Menu 

### Notes
- **N**: Open Notes 

### Other (CommandKeys)
- **B**: Execute command `/cc shards WEAPON` 
- **CTRL + B**: Execute command `/cc shards TOOL` 
- **SHIFT + B**: Execute command `/cc shards ARMOR` 
- **X**: Execute command `/cc moblexicon` 
- **Y**: Execute command `/cc attributes` 

### Xaero's Minimap
- **+**: Zoom In Minimap 
- **-**: Zoom Out Minimap 
- **R**: Enlarge Minimap 
- **U**: Open Waypoint Screen 
- **ALT + U**: Toggle On-Map Waypoints 
- **CTRL + M**: Toggle Minimap 
- **CTRL + U**: Toggle In-World Waypoints 

### Xaero's World Map
- **M**: Open World Map 

### Zoomify
- **C**: Zoom (Hold) 

## Performance notes

The “Entity Radar” feature is disabled by default because it caused strong framerate drops on some devices.   
If you want to enable it manually, open: `Mod Menu -> Xaero's Minimap -> Entity Radar Settings -> Display Radar`.   
When enabled, players and NPCs will be shown on the minimap.
