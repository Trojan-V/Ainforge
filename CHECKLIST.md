## Before each `git commit`

The following steps should be executed before every `git commit`.

1. The version in the `pack.toml` should match the version in the Minecraft window title configured in `config/defaultoptions/extra/config/fancymenu/options.txt` within the property `S:custom_window_title`.

## Before each `git push --tags`

1. The version in the `pack.toml` should match the version of the tag that is pushed to the remote.
